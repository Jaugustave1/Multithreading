Description:

This backend project for Dogether implements a servlet-based calendar system using Java and Apache Tomcat. The project includes servlet configuration, JAR library setup, and Eclipse integration.

Main Files:
- CalendarServlet.java: Main servlet for calendar task handling
- javax.servlet-api-4.0.1.jar: Required library for servlet support
- web.xml: Deployment descriptor for servlet configuration
- doc/: Contains documentation for deployment, requirements, and specifications

Instructions:
Refer to the deployment.txt file in the `doc/` folder for complete setup and testing instructions.
